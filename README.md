# 微信商城

## 技术选型
### 前端
* 前端框架: vue
* UI库: element-ui
* 数据可视化: echarts
* 富文本编辑器: ueditor
* 打包工具: webpack
* 构建工具: gulp

### 后端
* web服务器: nginx
* 后台: node.js
* 框架: express
* 模板引擎: handlebars
* 持久层: sequelize
* 数据库: mysql
